*Because I am currently using free [dynos](https://www.heroku.com/dynos) from *[Heroku](https://heroku.com), the site load time is not an accurate reflection of the actual load speed of the site. *But Why?*
> **Dyno sleeping**
If an app has a free web dyno, and that dyno receives no web traffic in a 30-minute period, *it will sleep*.
Free web dynos do not consume free dyno hours while sleeping.
If a sleeping web dyno receives web traffic, it will become active again after a short delay (assuming your account has free dyno hours available).f
